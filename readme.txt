== Descruption ==

The Simple markers map Plugin Gutenberg Block is a WordPress plugin that enables you to easily add customizable maps to your posts and pages using the Gutenberg editor.

1. Easy Integration: Add Maps directly from the Gutenberg editor.
2. Customizable Settings: Adjust map settings such as zoom level, map type, and marker positions.
3. Multiple Markers: Add multiple markers to highlight various locations on a single map.
4. User-Friendly Interface: Intuitive controls and settings for effortless map customization.


== Installation ==

Go to your WordPress dashboard.
Navigate to Plugins > Add New > Upload Plugin.
Choose the downloaded zip file and click Install Now.
Activate the Plugin.


== Usage ==

In the Gutenberg editor, click the + button to add a new block.
Search for "Simple markers map" and add the block to your post or page.
Customize Your Map.
